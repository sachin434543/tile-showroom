# Jai Gurudev Ceramics & Hardware

A premium luxury tile showroom website built with Next.js 15, React, TypeScript, Tailwind CSS, Three.js, GSAP, Framer Motion, and Supabase.

## Features

- Luxury animated landing page
- Smooth scrolling with Lenis
- 3D hero section
- Premium tile catalog
- Search and filters
- Gallery
- Testimonials
- Contact & inquiry form
- Supabase database integration
- Admin dashboard (in progress)

## Tech Stack

- Next.js 15
- React
- TypeScript
- Tailwind CSS
- Framer Motion
- GSAP
- Three.js
- React Three Fiber
- Supabase
- PostgreSQL
